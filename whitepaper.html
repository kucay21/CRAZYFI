Crazy Snake Whitepaper

1. Introduction

Crazy Snake adalah game Web3 berbasis blockchain Base Network yang menggabungkan keseruan permainan klasik snake dengan teknologi on-chain. Pemain bisa mengendalikan ular, mengumpulkan item, dan mendapatkan reward dalam bentuk token Crazy Snake Token (CST).

2. Gameplay

Konsep utama Crazy Snake adalah memberikan pengalaman bermain sederhana namun kompetitif. Pemain dapat:

  • Mengendalikan ular untuk mengumpulkan makanan.
  • Semakin panjang ular, semakin besar potensi reward.
  • Menggunakan item khusus dari marketplace untuk memperkuat gameplay.
  • Bersaing di leaderboard global untuk hadiah mingguan.

3. Tokenomics

Total Supply: 36,000,000 CST

  • 15,000,000 CST (41.6%) → Tim & Founder
  • 8,000,000 CST (22.2%) → Staking Reward
  • 7,000,000 CST (19.4%) → Game Reward
  • 4,000,000 CST (11.1%) → Community Incentive
  • 2,000,000 CST (5.5%) → Liquidity & Listing

4. Utility

  • Digunakan untuk membeli item dan skin di marketplace.
  • Dapat di-stake untuk mendapatkan CST tambahan.
  • Sebagai alat pembayaran internal ekosistem Crazy Snake.
  • Voting governance untuk arah pengembangan project.

5. Roadmap

  • Q3 2025: Launch Crazy Snake Beta + Staking Live
  • Q4 2025: Marketplace Rilis + Event Turnamen
  • 2026: Fitur NFT Skin, Guild System, & Tournament On-Chain

6. Conclusion

Crazy Snake bukan sekadar game, melainkan ekosistem play-to-earn yang transparan, sustainable, dan sepenuhnya berjalan di blockchain Base. Dengan token CST, komunitas dapat menikmati hiburan sekaligus peluang finansial dalam satu platform.
